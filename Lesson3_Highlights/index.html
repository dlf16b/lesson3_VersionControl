Version Control Why do we need it?

When multiple team members are working together and want to edit a file at the same time, problems may arise causing possible overwrites and inconsistencies in files. Version control creates one main repository that enables team members to check out files and edit them. This helps to solve the problem of overwriting.

Version Control Systems (VCS)

Version Control Systems automatically take note of who changed the file, when it was changed, what about the file is new or different, and if the file has been merged with another. These systems are also capable of reverting the file back to a previous version if the user wishes. VCS also contain control histories which facillitate file editing for the user.

Remote/Local Repositories What is the difference?

A repository is essentially where files and information can be stored. In this lesson we learned about both remote repositories and local repositories. A remote repository resides on a remote server that is shared among multiple team members or individuals. A local repository however, resides on a local machine of an individual user and can be created in one of the following two ways: creating a brand new repository, or cloning an exsisting remote repository onto a local machine.

GIT What and Why?

GIT is a distributed VCS which means that every user has a copy of the repository on their own machines. There is no single point of failure and processes tend to execute much more quickly. GIT is benefitial in the fact that it prevents unintentional overwrite.

GIT Commands 3 Basic Steps

There are a number of GIT commands that help users to navigate and operate within version control systems such as GIT. When it comes to basic GIT commands there are 3 basic steps to transfer a local project to a remote repository. The first step is to ADD. This adds the working copy of the file to something comparable to a "cache". The next step would be to COMMIT. This commits the copy to the local repository. And finally, PUSH that copy to the remote repository. These steps were crucial in completing the lesson 3 assignment.
